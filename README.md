# Kaushik Portfolio (Multi-Page)

Animated multi-page portfolio tailored to your resume.

## Run

```bash
cd /Users/kaushik/Desktop/Portfolio
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Files

- `index.html` - home page
- `about.html` - about page
- `experience.html` - experience page
- `projects.html` - GitHub projects page
- `skills.html` - backend + frontend stack
- `contact.html` - contact page
- `styles.css` - new visual system and animations
- `script.js` - particle background, reveal transitions, counters, card tilt
